# RiverEdge 系统后端 (tree-root)

**系统级后端 API 和核心功能** - 面向租户内用户和业务操作

## 🎯 职责范围

**系统级功能** - 租户内部的用户和业务管理：
- ⏳ 用户认证和管理（租户内用户）
- ⏳ 角色和权限管理
- ⏳ 用户业务操作
- ⏳ 租户内数据隔离

## 📋 当前状态

**目前tree-root为空**，等待系统级功能开发。

系统级功能目前由soil统一处理，包括：
- 用户模型（User）- 同时处理平台管理和租户内用户
- 用户认证服务
- 用户管理服务

## 🚧 未来规划

当系统级功能复杂化时，考虑将以下功能从soil迁移到tree-root：
- 租户内用户管理（普通用户和组织管理员）
- 租户内角色权限管理
- 租户内业务API
- 租户内仪表盘
