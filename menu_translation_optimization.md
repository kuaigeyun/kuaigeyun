# 菜单英文翻译优化方案

## 优化原则
1. 保持行业术语准确性
2. 减少字符长度，避免菜单显示不全
3. 使用行业常用缩写（如 FG=Finished Goods, SOP=Standard Operating Procedures）
4. 在分组菜单下，子菜单可以省略"Management"等重复词汇

## 需要优化的翻译

### kuaizhizao 应用

#### 生产执行 (Production Execution)
- 'Work Reporting Statistics' (24字符) → 'Work Stats' (10字符) 或 'Reporting Stats' (15字符)
- 'Material Shortage Exceptions' (29字符) → 'Material Shortage' (18字符)
- 'Delivery Delay Exceptions' (26字符) → 'Delivery Delay' (16字符)
- 'Exception Statistics' (20字符) → 'Exception Stats' (15字符)
- 'Exception Processing' (20字符) → 'Exception Process' (17字符)
- 'Outsource Work Orders' (21字符) → 'Outsource WOs' (15字符) 或保持

#### 质量管理 (Quality Management)
- 'Finished Goods Inspection' (25字符) → 'FG Inspection' (14字符)

#### 成本管理 (Cost Management)
- 'Cost Calculations' (17字符) → 'Cost Calc' (10字符)
- 'Cost Optimization' (18字符) → 'Cost Optimize' (14字符)

#### 设备管理 (Equipment Management)
- 'Equipment Management' (21字符) → 'Equipment' (10字符) - 已在Equipment Management分组下
- 'Maintenance Reminders' (21字符) → 'Maint. Reminders' (18字符) 或 'Maintenance Alerts' (20字符)

#### 财务管理 (Finance Management)
- 'Accounts Payable' (16字符) → 'Payables' (9字符)
- 'Accounts Receivable' (19字符) → 'Receivables' (11字符)

#### 报表分析 (Reports)
- 'Reports & Analysis' (18字符) → 'Reports' (8字符)

#### 仓储管理 (Warehouse Management)
- 'Warehouse Management' (21字符) → 'Warehouse' (10字符) - 已在Warehouse Management分组下
- 'Finished Goods Inventory' (24字符) → 'FG Inventory' (13字符)
- 'Customer Material Registration' (30字符) → 'Customer Material' (17字符) 或 'Cust. Material' (15字符)
- 'Barcode Mapping Rules' (21字符) → 'Barcode Rules' (14字符)
- 'Document Efficiency' (19字符) → 'Doc Efficiency' (15字符)
- 'Replenishment Suggestions' (25字符) → 'Replenishment' (14字符)

### master-data 应用

#### 工厂数据 (Factory)
- 'Factory Modeling' (16字符) → 'Factory Model' (14字符) 或保持
- 'Plant Management' (18字符) → 'Plants' (6字符)
- 'Workshop Management' (20字符) → 'Workshops' (10字符)
- 'Production Line Management' (26字符) → 'Production Lines' (17字符)
- 'Workstation Management' (23字符) → 'Workstations' (13字符)

#### 物料数据 (Materials)
- 'Material Management' (20字符) → 'Materials' (9字符)
- 'Material List BOM' (18字符) → 'BOM List' (9字符) 或 'Material BOM' (12字符)
- 'Material Code Mapping' (22字符) → 'Code Mapping' (13字符)
- 'Batch Management' (17字符) → 'Batches' (8字符)
- 'Serial Number Management' (24字符) → 'Serial Numbers' (14字符) 或 'Serials' (8字符)

#### 工艺数据 (Process)
- 'Process Management' (19字符) → 'Process' (7字符)
- 'Defect Type Management' (23字符) → 'Defect Types' (13字符)
- 'Standard Operating Procedures' (30字符) → 'SOP' (3字符) 或 'Procedures' (11字符)

#### 绩效数据 (Performance)
- 'Performance Management' (22字符) → 'Performance' (11字符)
- 'Skill Management' (17字符) → 'Skills' (7字符)
