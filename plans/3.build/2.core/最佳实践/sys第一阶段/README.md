**预计时间**：1-2 周  
**状态**：🚧 建设中

**文件位置**：
- 后端：`riveredge-backend/src/tree-root/api/roles/`, `api/permissions/`

1. ✅ 创建数据库模型（Role、Permission、RolePermission、UserRole）

4. ✅ 创建 API（路由）
