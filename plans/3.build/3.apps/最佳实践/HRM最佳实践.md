# HRM - 人力资源管理最佳实践

## 📋 模块概述

**快格轻HRM (kuaihrm)** - 人力资源管理模块，聚焦人力资源业务逻辑，数据管理归主数据。

## 🎯 设计原则：小快轻准

- ✅ **员工异动规范化**：规范化的员工异动流程
- ✅ **考勤管理自动化**：自动化的考勤管理，提升效率
- ✅ **薪资计算准确性**：准确的薪资计算，支持复杂薪资结构
- ✅ **移动端应用**：移动端打卡、请假等应用

## 🏆 核心实践

### 1. 员工异动管理

#### 1.1 入职流程
- **入职申请**：入职申请、申请审批
- **入职办理**：入职办理、办理流程
- **入职确认**：入职确认、确认记录

#### 1.2 离职流程
- **离职申请**：离职申请、申请审批
- **离职办理**：离职办理、办理流程
- **离职交接**：离职交接、交接确认

#### 1.3 员工异动
- **异动类型**：调岗、晋升、降职等异动类型
- **异动流程**：异动流程、流程审批
- **异动记录**：异动记录、异动历史查询

### 2. 考勤管理

#### 2.2 打卡记录
- **打卡登记**：打卡登记、打卡方式（GPS、WiFi、蓝牙等）
- **打卡统计**：打卡统计、统计报表
- **异常处理**：打卡异常处理、异常分析

#### 2.3 请假加班
- **请假申请**：请假申请、申请审批
- **加班申请**：加班申请、申请审批
- **审批流程**：审批流程、流程跟踪

#### 2.4 考勤统计
- **考勤报表**：考勤报表生成、报表分析
- **考勤分析**：考勤分析、分析报表
- **考勤查询**：考勤查询、查询导出

### 3. 薪资管理

#### 3.4 薪资发放

#### 4.1 排班计划

- **计划调整**：排班计划调整、调整审批

## 🔗 数据来源

- ← **master-data**：员工档案、组织架构、部门、岗位、技能、资质、假期类型

## 🔌 集成接口

- → **MES**：人员排班、人员技能、工时数据
- → **EAM**：操作人员资质、技能要求
- → **QMS**：质量培训、资质管理
- → **APS**：人员可用性、人员产能
- ← **EHS**：员工健康档案
- ← **master-data**：员工基础信息、组织架构、部门、岗位、技能、资质
