# SRM - 采购管理最佳实践

## 📋 模块概述

**快格轻SRM (kuaisrm)** - 采购管理模块，聚焦采购业务逻辑，数据管理归主数据。

## 🎯 设计原则：小快轻准

- ✅ **采购流程标准化**：标准化的采购流程，提升采购效率
- ✅ **供应商协同**：供应商门户，实现采购协同
- ✅ **委外管理规范化**：规范化的委外管理流程
- ✅ **供应商绩效管理**：基于数据的供应商绩效评估

## 🏆 核心实践

### 1. 采购订单管理

#### 1.1 需求转订单
- **自动转单**：MRP需求自动转采购订单
- **批量转单**：支持批量转单，提升效率
- **转单规则**：转单规则配置（供应商选择、价格选择等）
- **转单校验**：转单数据校验、转单确认

#### 1.2 订单审批
- **多级审批**：多级审批流程，审批流程可配置
- **条件审批**：基于订单金额、供应商等条件自动选择审批流程
- **审批记录**：审批意见记录、审批结论记录
- **审批提醒**：审批任务自动提醒

#### 1.3 订单跟踪
- **状态跟踪**：订单状态跟踪、状态变更通知
- **交期跟踪**：订单交期跟踪、交期预警
- **到货跟踪**：到货跟踪、到货确认
- **订单看板**：订单看板展示，支持多维度筛选

#### 1.4 订单变更
- **变更申请**：订单变更申请，说明变更原因
- **变更审批**：变更审批流程，重要变更需审批
- **变更通知**：变更通知供应商、变更通知相关模块
- **变更记录**：变更记录、变更历史查询

### 2. 委外订单管理

#### 2.1 委外需求
- **需求接收**：MRP委外需求自动转委外订单
- **供应商选择**：委外供应商选择、供应商评估
- **订单创建**：委外订单创建、订单确认

#### 2.2 委外跟踪
- **进度跟踪**：委外加工进度跟踪、进度更新
- **交期管理**：委外交期管理、交期预警
- **异常处理**：委外异常处理、异常跟踪
- **进度可视化**：委外进度可视化展示

#### 2.3 委外协同
- **供应商门户**：委外供应商门户，支持进度更新、异常反馈
- **协同沟通**：协同沟通、问题反馈
- **协同计划**：协同计划制定、计划执行跟踪

### 3. 供应商协同

#### 3.1 供应商门户
- **订单查询**：订单查询、订单详情查看
- **交货管理**：交货计划、交货执行、交货确认
- **对账管理**：对账单查询、对账确认
- **协同沟通**：协同沟通、问题反馈

#### 3.2 预测共享
- **需求预测**：需求预测共享，支持供应商备货
- **预测准确性**：预测准确性分析、预测优化
- **协同计划**：基于预测的协同计划制定

#### 3.3 协同计划
- **计划制定**：协同计划制定、计划确认
- **计划执行**：计划执行跟踪、执行反馈
- **计划调整**：计划调整、调整通知

### 4. 供应商评估

#### 4.1 评估维度
- **质量维度**：来料合格率、质量稳定性
- **交期维度**：交期准确率、交期稳定性
- **价格维度**：价格竞争力、价格稳定性
- **服务维度**：服务响应、服务态度

#### 4.2 评估数据
- **自动评估**：基于订单执行数据自动评估
- **评估周期**：月度评估、季度评估、年度评估
- **评估记录**：评估记录、评估历史查询

#### 4.3 评估结果
- **评估等级**：评估等级（A、B、C、D等）
- **评估报告**：评估报告生成、报告分析
- **改进建议**：改进建议、改进跟踪

### 5. 采购分析

#### 5.1 成本分析
- **成本趋势**：采购成本趋势分析
- **成本结构**：成本结构分析、成本占比分析
- **成本对比**：供应商成本对比、历史成本对比

#### 5.2 供应商分析
- **绩效排名**：供应商绩效排名、排名变化趋势
- **风险分析**：供应商风险分析、风险预警
- **供应商画像**：供应商画像分析

#### 5.3 委外分析
- **委外成本**：委外成本分析、成本结构分析
- **委外效率**：委外效率分析、效率提升建议
- **委外质量**：委外质量分析、质量改进建议

## 🔗 数据来源

- ← **master-data**：供应商基础信息、委外供应商基础信息、物料信息

## 🔌 集成接口

- ← **MRP**：采购需求、委外需求
- ← **SCM**：协同采购需求、预测共享
- → **WMS**：到货通知、入库指令、委外发料指令、委外回收指令
- → **QMS**：来料检验任务、委外来料检验任务
- → **财务**：采购发票、付款管理、委外加工费结算
- → **SCM**：采购执行数据、供应商绩效
- → **MES**：委外进度、委外完成状态
- ← **master-data**：供应商信息、委外供应商信息、物料信息

- 移动端供应商协同

## 💡 技术建议

### 1. 供应商选择算法
- 基于多维度评估的供应商选择算法
- 考虑质量、交期、价格、服务等权重
- 支持供应商推荐

### 2. 供应商门户设计
- 响应式设计，支持PC和移动端
- 实时数据同步
- 安全认证和权限控制

### 3. 供应商绩效评估模型
- 基于历史数据的绩效评估模型
- 支持自定义评估维度和权重
- 支持绩效趋势分析

### 4. 委外进度跟踪
- 实时进度更新机制
- 进度异常预警
- 进度可视化展示

## 📊 关键指标

- 采购订单及时率：按时到货订单占比
- 供应商绩效：供应商综合绩效评分
- 委外效率：委外订单按时完成率
- 采购成本节约：采购成本节约率
- 供应商协同度：供应商协同响应度
