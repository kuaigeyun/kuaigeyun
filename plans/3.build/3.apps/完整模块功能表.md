# 快格轻工厂全流程管理系统 - 完整模块功能表

## 🎯 设计原则：小快轻准

**快格轻工厂全流程管理系统**遵循**"小快轻准"**设计原则，结合最佳实践，实现数据与业务分离的清晰架构：

### 核心设计思想

1. **小（Small）**：功能精简，聚焦核心
   - 每个模块聚焦核心业务逻辑，避免功能冗余
   - 模块职责清晰，边界明确
   - 快速响应业务需求

2. **快（Fast）**：快速响应，敏捷开发
   - 模块化设计，支持快速迭代
   - 标准化接口，降低集成成本
   - 轻量级架构，提升系统性能

3. **轻（Light）**：轻量级，避免过度设计
   - 避免功能堆砌，保持系统简洁
   - 按需加载，按需扩展
   - 降低系统复杂度

4. **准（Precise）**：精准定位，准确满足需求
   - 数据与业务分离，职责清晰
   - 精准的功能定位，避免功能重叠
   - 准确的数据流向，确保数据一致性

### 数据与业务分离原则

**数据管理统一归口（主数据管理）**：
- ✅ 所有基础数据在主数据管理统一维护
- ✅ 客户、供应商、设备、工装模具、产品、员工等基础档案统一管理
- ✅ 物料、BOM、工艺路线、SOP等工艺数据统一管理
- ✅ 组织架构、部门、岗位等组织数据统一管理
- ✅ 数据变更统一通知，确保数据一致性

**业务模块聚焦业务逻辑**：
- ✅ 业务模块只负责业务流程和业务逻辑
- ✅ 订单管理、工单管理、检验任务等业务流程
- ✅ 状态流转、审批流程、执行跟踪等业务逻辑
- ✅ 从主数据获取基础数据，不维护数据档案

**数据流向清晰**：
- 主数据管理 → 业务模块：提供基础数据查询
- 业务模块 → 主数据管理：数据使用统计、数据关联关系
- 业务模块 ↔ 业务模块：业务流程数据交互

- ✅ 统一数据标准：所有基础数据在主数据管理统一维护

### 3. 快格轻MRP (kuaimrp) - 物料需求 ⚠️
**核心功能：**
1. **MRP需求计算**（全局计划）：
   - 基于所有需求源（销售订单、预测单、发货单等）进行全局物料需求计算
   - 分层处理逻辑，逐级分解物料需求
   - 生产计划和采购计划同时产出
   - 适用于库存数据准确、供应链管理成熟的环境
2. **LRP需求计算**（批次计划）：
   - 按订单进行物料需求分解，支持订单合并成计划批次
   - 按单处理逻辑，清晰跟踪需求来源
   - 可以先计算生产计划，再计算采购计划
   - 适用于订单驱动生产、中小企业场景
3. **采购计划**：生成采购建议、采购计划优化
4. **库存分析**：安全库存、库存预警、缺料分析
5. **需求追溯**：需求来源追溯、需求变更影响分析、全阶供需关系查看

**集成接口：**
- ← CRM：销售订单
- ← MES：生产计划、工单需求
- → SRM：采购需求推送、委外需求、委外计划
- → SCM：需求计划、物料需求
- → WMS：库存查询、库存预留
- ← master-data：BOM、物料信息

**移动端支持：**
- 移动端需求查看、移动端缺料预警、移动端需求追溯、移动端计划查看

**说明：**
- **MRP（Material Requirements Planning）**：物料需求计划，全局性计划方法，针对所有物料的供需进行整体平衡
  - **适用模式**：MTS（Make to Stock，按库存生产）
  - **特点**：基于预测和销售订单，全局平衡物料需求，适用于库存驱动生产
- **LRP（Lot Requirements Planning）**：批次需求计划，局部计划方法，按订单进行物料需求分解，便于跟踪和插单管理
  - **适用模式**：MTO（Make to Order，按订单生产）
  - **特点**：按订单分解需求，清晰跟踪需求来源，适用于订单驱动生产

**核心实践**：

   - 权限继承：组织架构权限继承、角色权限继承

   - 异常监控：异常日志自动收集、异常告警、异常趋势分析
   - 接口监控：接口调用统计、接口健康检查、接口性能分析

4. **数据安全**：
   - 数据加密：敏感数据加密存储、传输加密
