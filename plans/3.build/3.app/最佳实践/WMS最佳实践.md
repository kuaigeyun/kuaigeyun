# WMS - 仓库管理最佳实践

## 📋 模块概述

**快格轻WMS (kuaiwms)** - 仓库管理模块，聚焦仓库业务逻辑，数据管理归主数据。

## 🎯 设计原则：小快轻准

- ✅ **库存实时准确**：实时库存更新，确保库存准确性
- ✅ **入库出库标准化**：标准化的入库出库流程，提升效率
- ✅ **库位管理精细化**：精细化的库位管理，提升空间利用率
- ✅ **移动端操作优先**：仓库现场操作优先支持移动端

## 🏆 核心实践

### 1. 库存管理

#### 1.1 实时库存
- **库存查询**：实时库存查询、库存统计
- **库存预警**：库存预警（低库存、高库存、呆滞库存）
- **库存分析**：库存周转率、库存结构、库存成本分析
- **库存可视化**：库存可视化展示、库存看板

#### 1.2 库存调整
- **调整申请**：库存调整申请，说明调整原因
- **调整审批**：调整审批流程，重要调整需审批
- **调整执行**：调整执行、调整确认
- **调整记录**：调整记录、调整历史查询

#### 1.3 库存盘点
- **盘点计划**：盘点计划制定、盘点范围确定
- **盘点执行**：盘点执行、盘点数据录入
- **盘点差异**：盘点差异分析、差异处理
- **盘点报告**：盘点报告生成、报告分析

### 2. 入库管理

#### 2.1 入库流程
- **入库申请**：入库申请、入库计划
- **入库执行**：入库执行、入库确认
- **入库质检**：入库质检、质检放行
- **入库记录**：入库记录、入库历史查询

#### 2.2 入库方式
- **采购入库**：采购到货入库、入库确认
- **生产入库**：生产完工入库、入库确认
- **退货入库**：退货入库、入库确认
- **委外入库**：委外完工入库、入库确认

#### 2.3 入库优化
- **任务分配**：入库任务分配、任务优化
- **路径优化**：入库路径优化、路径规划
- **效率提升**：入库效率分析、效率提升建议

### 3. 出库管理

#### 3.1 出库流程
- **出库申请**：出库申请、出库计划
- **出库执行**：出库执行、出库确认
- **出库记录**：出库记录、出库历史查询

#### 3.2 出库方式
- **销售出库**：销售出库、出库确认
- **生产领料**：生产领料、领料确认
- **调拨出库**：调拨出库、出库确认
- **委外发料**：委外发料、发料确认

#### 3.3 拣货优化
- **任务分配**：拣货任务分配、任务优化
- **路径优化**：拣货路径优化、路径规划
- **批量拣货**：批量拣货、拣货效率提升

### 4. 委外物料业务

#### 4.1 委外发料
- **发料单**：委外发料单创建、发料计划
- **发料跟踪**：发料跟踪、发料确认
- **发料记录**：发料记录、发料历史查询

#### 4.2 委外回收
- **回收单**：委外回收单创建、回收计划
- **回收跟踪**：回收跟踪、回收确认
- **回收记录**：回收记录、回收历史查询

#### 4.3 委外余料
- **余料统计**：委外余料统计、余料分析
- **余料回收**：余料回收、回收确认
- **余料处理**：余料处理、处理记录

### 5. 库位管理

#### 5.1 库位分配
- **自动分配**：基于规则的自动库位分配
  - 按物料属性分配（温度、湿度要求等）
  - 按使用频率分配（高频物料就近）
  - 按体积重量分配（大件物料合理分配）
- **手动分配**：手动库位分配、分配调整
- **分配优化**：库位分配优化建议

#### 5.2 库位优化
- **使用频率分析**：库位使用频率分析
- **优化建议**：基于使用频率的库位优化建议
- **优化执行**：库位优化执行、优化效果评估

#### 5.3 库位盘点
- **盘点计划**：库位盘点计划、盘点范围
- **盘点执行**：库位盘点执行、盘点数据录入
- **盘点差异**：库位盘点差异分析、差异处理

### 6. 内部物流配送

#### 6.1 车间物料配送
- **Line Feeding**：准时化配送，支持看板拉动
- **配送计划**：基于生产计划的配送计划制定
- **配送执行**：配送执行、配送确认
- **配送跟踪**：配送跟踪、配送状态更新

#### 6.2 配送优化
- **任务分配**：配送任务分配、任务优化
- **路径优化**：配送路径优化、路径规划
- **配送效率**：配送效率分析、效率提升建议

## 🔗 数据来源

- ← **master-data**：仓库、库区、库位信息、物料信息

## 🔌 集成接口

- ← **SRM**：采购到货、入库指令、委外发料指令、委外回收指令
- ← **MES**：生产领料、生产入库、配送需求
- → **CRM**：发货管理、备件库存
- → **MRP**：库存数据、可用库存
- → **SCM**：全局库存数据
- → **财务**：库存成本、库存价值
- ← **master-data**：仓库、库位信息、物料信息

## 📱 移动端支持

- 移动端入库（扫码入库、批量入库）
- 移动端出库（扫码出库、批量出库）
- 移动端盘点（扫码盘点、盘点数据录入）
- 移动端配送（配送任务、配送确认）
- 移动端拣货（拣货任务、拣货确认）
- 移动端委外发料确认
- 移动端委外回收确认

## 💡 技术建议

### 1. 库存实时更新机制
- 使用事件驱动机制实现库存实时更新
- 支持库存预留、库存释放
- 库存变更记录审计日志

### 2. 库位分配算法
- 基于规则的库位分配算法
- 考虑物料属性、使用频率、体积重量等因素
- 支持库位分配优化

### 3. 拣货路径优化算法
- 使用TSP（旅行商问题）算法优化拣货路径
- 考虑库位距离、拣货顺序等因素
- 支持批量拣货优化

### 4. 移动端离线支持
- 支持离线入库、离线出库
- 网络恢复后自动同步
- 数据冲突处理机制

- 出库效率：出库处理效率
- 库位利用率：库位空间利用率
- 配送及时率：配送按时完成率
