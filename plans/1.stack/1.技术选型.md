**Docker + Docker Compose**

- **开发环境**: Docker Compose
- **生产环境**: Docker + Kubernetes
- **Redis 服务器**: ⚠️ **必须使用 Redis 6.2.6+** 服务器版本（BSD 3-Clause 许可证）
  - **Docker 镜像**: `redis:6.2.6` 或 `redis:6.2`（推荐使用具体版本号）
  - **部署示例**: `docker run -d -p 6379:6379 --name redis redis:6.2.6`
  - **Docker Compose 示例**: `image: redis:6.2.6`
  - **⚠️ 禁止使用**: Redis 7.0+（RSAL 许可证限制商业使用）

### 传统部署

- **后端**:
  - `uvicorn` + `gunicorn` (ASGI 服务器)
  - `nginx` (反向代理)
  - `supervisor` (进程管理)
  - `kkfileview` (文件预览服务，Docker 部署)
- **前端**:
  - `nginx` (静态文件服务)
