# RiverEdge SaaS 多租户框架 - 品牌VI一致性规范

## 📋 概述

本文档定义 **RiverEdge SaaS 多租户框架 (RiverEdge SaaS Multi-tenant Framework)** 的**统一品牌视觉识别（VI）规范**，基于 Ant Design Pro 的 B 端产品设计最佳实践，确保整个产品的视觉风格统一、专业、现代。

**创建日期**：2025-11-17

## 🎯 设计原则

1. **一致性**：所有界面元素遵循统一的设计规范
2. **专业性**：符合 B 端产品的专业形象
3. **可读性**：确保信息清晰易读，提升用户体验
4. **可访问性**：遵循 WCAG 2.1 无障碍设计标准
5. **品牌识别**：保持品牌视觉元素的一致性

## 📝 字体规范

**基础字号**：`14px`（Ant Design 默认）

**字号层级**：

**标题字号**：

```typescript

  normal: 400,  // 常规
```

## 📏 间距规范

### 基础间距单位

**基础单位**：`8px`（Ant Design 使用 8px 网格系统）

**间距值**：
```typescript
const spacing = {
  xs: 4,   // 4px - 极小间距
  sm: 8,   // 8px - 小间距
  md: 16,  // 16px - 中等间距（默认）
  lg: 24,  // 24px - 大间距
  xl: 32,  // 32px - 特大间距
  xxl: 48, // 48px - 超大间距
};
```

### 组件间距

**内边距（Padding）**：
- 按钮：`8px 16px`（垂直 8px，水平 16px）
- 输入框：`4px 12px`
- 卡片：`24px`
- 表格单元格：`16px 16px`

**外边距（Margin）**：
- 组件之间：`16px` 或 `24px`
- 区块之间：`24px` 或 `32px`
- 页面边距：`24px`

### 布局间距

**页面布局**：
- 页面内边距：`24px`
- 内容区域最大宽度：`1200px`（居中）
- 侧边栏宽度：`256px`（收起时 `80px`）
- 头部高度：`64px`

## 🧩 组件样式规范

**类型**：
- Primary：主色背景，白色文字
- Default：白色背景，灰色边框
- Dashed：虚线边框
- Text：文字按钮
- Link：链接按钮

### 输入框（Input）

**尺寸**：
- 大：高度 `40px`
- 中（默认）：高度 `32px`
- 小：高度 `24px`

**状态样式**：
- 默认：边框 `#d9d9d9`
- 悬停：边框 `#40a9ff`

**边框**：

**斑马纹**：
- 偶数行背景：`#fafafa`（可选）

## 📐 布局规范

### 栅格系统

**24 栅格系统**（Ant Design 标准）：
- 总列数：`24`
- 栅格间隔：`16px` 或 `24px`
- 响应式断点：见下方

**使用示例**：
```tsx
<Row gutter={16}>
  <Col span={12}>50% 宽度</Col>
  <Col span={12}>50% 宽度</Col>
</Row>

<Row gutter={24}>
  <Col xs={24} sm={12} md={8} lg={6}>
    响应式列
  </Col>
</Row>
```

### 响应式断点

```typescript
const breakpoints = {
  xs: 480,   // 超小屏（手机）
  sm: 576,   // 小屏（手机横屏）
  md: 768,   // 中屏（平板）
  lg: 992,   // 大屏（桌面）
  xl: 1200,  // 超大屏（大桌面）
  xxl: 1600, // 超超大屏（超大桌面）
};
```

**布局适配**：
- `xs`：单列布局，侧边栏收起
- `sm`：单列布局，侧边栏收起
- `md`：双列布局，侧边栏可展开
- `lg+`：多列布局，侧边栏展开

### 页面布局

**标准布局结构**：
```
┌─────────────────────────────────────┐
│ Header (64px)                        │
├──────────┬──────────────────────────┤
│          │                          │
│ Sidebar  │  Content Area            │
│ (256px)  │  (padding: 24px)         │
│          │                          │
│          │                          │
└──────────┴──────────────────────────┘
```

**内容区域**：
- 最大宽度：`1200px`（居中）
- 内边距：`24px`
- 背景色：`#f0f2f5`
